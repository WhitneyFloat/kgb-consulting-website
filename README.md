# KGB Consulting Website

**Kelly Gill-Braxton — Fractional Head of Sales Compensation**

A premium, editorial single-page consulting landing page targeting CROs, VPs of Sales, and CFOs at Series A–C companies.

## Features
- Animated iridescent hero blob
- Liquid glass card hover effects
- Scroll-reveal animations via IntersectionObserver
- Responsive mobile-first design
- Inline contact form with client-side state

## Preview locally
```bash
ruby -run -e httpd . -p 8080
```
Then open http://localhost:8080

## Tech Stack
- Vanilla HTML / CSS / JS (single file)
- Google Fonts: Cormorant Garamond + DM Sans
- No external JS libraries
